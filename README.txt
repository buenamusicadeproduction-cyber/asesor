Servidor Gemini (para Asesor Personal IA)

Instrucciones rápidas:
1) Obtén una API key de Google Generative AI / Gemini (Google AI Studio).
2) En Render o Railway, despliega este proyecto:
   - Build command: npm install
   - Start command: npm start
3) Define la variable de entorno GEMINI_API_KEY en el panel del servicio.
4) El endpoint estará en /api/interpret
